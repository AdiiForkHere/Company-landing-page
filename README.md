# 🌐 Company Landing Page

A modern and responsive **Company Landing Page** built using **HTML, CSS, and JavaScript**. This project represents a professional website design suitable for businesses, startups, or personal portfolios.

---

## 🚀 Features

- 📱 Fully Responsive Design (Mobile + Desktop)
- 🎨 Clean and Modern UI
- ⚡ Smooth Scrolling Effects
- 🧭 Navigation Bar with Sections
- 📦 Includes:
  - Home Section
  - About Section
  - Services Section
  - Contact Section
- 📩 Contact Form (Frontend Only)
- 💡 Interactive Elements using JavaScript

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the webpage  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Interactivity
